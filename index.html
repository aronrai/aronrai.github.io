About me

I am a software developer driven by the challenge of bridging the gap between complex backend logic and intuitive user interfaces. I believe that high-quality software should be as robust under the hood as it is seamless on the surface.

My work is centered on the MERN stack, utilizing MongoDB, Express, React, and Node.js to build full-scale applications. A primary example of this is Blogspot, a multi-user platform where I implemented secure JWT authentication and Cloudinary integration to handle real-world media and data management.

To ensure every project is production-ready, I rely on a modern toolkit including Zustand for state management and Tailwind CSS for responsive design. I use Postman for rigorous API testing and Git for version control, maintaining a workflow that prioritizes transparency and organization.

Ultimately, I focus on clean code and architectural integrity. My goal is to build software that isn't just functional, but performs efficiently and remains maintainable as it scales.
