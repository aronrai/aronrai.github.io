To-Do-List Web App

This To-Do List Web App is a simple yet efficient application built using HTML, CSS, and JavaScript. The app helps users stay organized by allowing them to manage tasks, add new items, mark them as completed, and delete them when done. With a user-friendly interface, this app ensures seamless interaction and a smooth user experience. HTML and CSS work together to create an intuitive and visually appealing layout, while JavaScript adds dynamic functionality and responsiveness, enabling smooth task management and real-time updates.

Weather Web App

This Weather Web App is a simple yet functional project designed to provide real-time weather updates and a 5-day forecast for any location. Built using HTML, CSS, and JavaScript, and powered by a OpenWeather API, the app displays key weather details like temperature, humidity, wind speed, and conditions in a clean and responsive interface. This project highlights fundamental front-end development skills and demonstrates how to integrate APIs to create practical and user-friendly applications.
