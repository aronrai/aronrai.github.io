About Me

I am a software developer with a deep focus on the JavaScript ecosystem and a passion for building applications that are as robust as they are intuitive. My work is centered on the MERN stack, where I navigate the complexities of architecting scalable databases in MongoDB and crafting secure, efficient APIs with Node.js and Express. I believe that high-quality software isn't just about making things work—it’s about creating systems that are production-ready and built to scale from the very first line of code.

On the frontend, I utilize React and Zustand to manage complex state with precision, ensuring a seamless experience for the end user. My technical process is guided by a commitment to clean code and architectural integrity, specifically when implementing features like secure JWT authentication and cloud-based media management. I rely on tools like Postman for rigorous testing and Git for version control, maintaining a workflow that prioritizes quality, security, and maintainability at every stage of development.
