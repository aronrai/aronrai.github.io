# Aron Rai • Portfolio

A minimalist, high-performance professional portfolio built with a "less is more" philosophy. This site serves as a central hub for my software development projects, technical skillset, and professional contact information.

## Tech Stack

- **HTML5**: Architecting a semantic, SEO-optimized foundation for maximum accessibility.
- **Tailwind CSS (v4)**: Leveraging a modern, utility-first engine to build a high-performance, minimalist UI.
- **Typed.js**: Enhancing interactive storytelling with dynamic, real-time typing animations.
- **Particles.js**: Implementing a lightweight canvas system for an immersive, ambient background.
- **Google Fonts**: Curating refined typography (Inter) for a clean and readable digital experience.
