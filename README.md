# aronrai.github.io

A minimalist, high-performance professional portfolio built with a "less is more" philosophy. This site serves as a central hub for my software development projects, technical skillset, and professional contact information.

## Tech Stack

- **HTML5**: Semantic and SEO-optimized structure.
- **Tailwind CSS (v4)**: Modern, utility-first styling for a clean UI.
- **Typed.js**: Dynamic typing effects for interactive storytelling.
- **Particles.js**: A lightweight JavaScript library for the ambient starry background.
- **Google Fonts**: Optimized typography (Inter).

## Key Features

- **About Me**: A concise overview of my journey as a software developer.
- **Projects**: Showcasing full-stack applications like **Blogspot** (MERN).
- **Contact**: Direct integration with Email and WhatsApp for quick inquiries.
- **Resume**: Easily accessible professional documentation.
- **Responsive**: Designed with a mobile-first approach to ensure a seamless experience on all devices.
